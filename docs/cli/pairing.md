# `zovsironclaw pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:

- Pairing flow: [Pairing](/start/pairing)

## Commands

```bash
zovsironclaw pairing list whatsapp
zovsironclaw pairing approve whatsapp <code> --notify
```
